# Get-ModuleImportOrder

> [!WARNING]
> This page is archived. `Get-ModuleImportOrder` is no longer exported by
> DLLPickle.

## SYNOPSIS

Archived command reference retained for migration guidance.

## SYNTAX

```powershell
Get-ModuleImportOrder [[-Name] <String[]>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

This command is no longer part of the exported DLLPickle command set.

Use `Get-ModulesWithVersionSortedIdentityClient` to evaluate modules by packaged
`Microsoft.Identity.Client.dll` version. Use `Get-ModuleImportCandidate` when
you need to determine the effective module selected by import resolution.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-ModulesWithVersionSortedIdentityClient -Name 'Az.Accounts','ExchangeOnlineManagement'
```

Returns modules ordered by packaged `Microsoft.Identity.Client.dll` version.
